    # 创建 OAuth 应用

    你可以创建和注册一个 OAuth 应用在你的个人账号下。在创建 OAuth 应用时，请记住保护你的个人隐私。

    Waning: OAuth 应用中仅使用你认为可以公开的信息，请避免使用敏感数据，例如内部URL

    # 第1步


    在任意页面的右上角，点击你的个人头像，然后点击 设置。




    # 第2步


    在左侧边栏中，点击 OAuth应用。




    # 第3步


    点击 新建。




    # 第4步


    在 “应用名”，输入你的应用名称




    # 第5步

    （可选）在“简介”中，输入用户将看到的应用程序的说明。

    # 第6步

    在 “主页”，输入你的应用网站点完整链接


    # 第7步

    在 “回调”，输入你的应用的回调地址

    # 第8步

    （可选）在“LOGO”中，上传你的应用 Logo。

    # 第9步

    点击 创建应用。
